# Examples

The files in `examples/synthetic/` are public-safe examples. They preserve the shape of the pipeline and recommendation outputs without using real activity exports, locations, account IDs, credentials, or health records.

Included examples:

- `activities.csv`: normalized synthetic activity rows
- `summary_weekly.csv`: synthetic weekly aggregation
- `pipeline-report.md`: shortened synthetic Markdown report
- `morning-recommendation.md`: synthetic morning decision output
- `notion-dashboard-preview.md`: synthetic Notion dry-run preview

These files are for review and documentation. They are not training advice.
